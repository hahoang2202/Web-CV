HyperText Markup Language hay còn gọi là HTML là ngôn ngữ đánh dấu tiêu chuẩn cho các tài liệu được thiết kế để hiển thị trong trình duyệt web. Nó có thể được trợ giúp bởi các công nghệ như CSS và các ngôn ngữ kịch bản giống như JavaScript

Cascading Style Sheet hay còn gọi là CSS là một ngôn ngữ lập trình được thiết kế rất đơn giản và dễ sử dụng. Mục tiêu của CSS là giúp đơn giản hóa quá trình tạo ra các website. CSS có nhiệm vụ thực hiện việc xử lý giao diện của một trang web. Có thể kể đến các yếu tố như màu sắc văn bản, hay khoảng cách giữa các đoạn, font chữ, hình ảnh, bố cục, màu nền,…

JavaScript là ngôn ngữ lập trình được nhà phát triển sử dụng để tạo trang web tương tác. Từ làm mới bảng tin trên trang mạng xã hội đến hiển thị hình ảnh động và bản đồ tương tác, các chức năng của JavaScript có thể cải thiện trải nghiệm người dùng của trang web. Là ngôn ngữ kịch bản phía máy khách, JavaScript là một trong những công nghệ cốt lõi của World Wide Web.
